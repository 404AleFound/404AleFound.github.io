题目：0002 两数相加

给你两个非空的链表，表示两个非负的整数。它们每位数字都是按照逆序的方式存储的，并且每个节点只能存储一位数字。请你将两个数相加，并以相同形式返回一个表示和的链表。你可以假设除了数字 0 之外，这两个数都不会以 0 开头，流程示意如下：

题解：

创建一个值 carry 用于记录进位信息。用一个 while 循环进行迭代累加，当 l1->val 或 l2->val 或 carry 中只要有一个值存在，则循环继续。在循环内部，用临时变量 value 记录当前步的加法值：

  • 如果 l1 当前节点存在，累加该值 value += l1->val；
  • 如果 l2 当前节点存在，累加该值 value += l2->val；
  • 如果 carry 非 0 ，累加该值 value += carry；

对累加后的 value 进行处理，更新 carry 值，保留十的余数：

  • carry = value / 10
  • value = value % 10

创建节点，存入本步的值，更新相关的迭代变量。
