Kotlin Types

2026-05-13 · updated 2026-05-19 · programming-languages , kotlin-note

Note

在 Kotlin 中，一切皆对象——你可以在任何变量上调用成员函数和属性。而本节主要介绍 Kotlin 中一些常用的基本类型，以下为 Kotlin 中常见的基本类型：

  • 数字及其无符号类型
  • 布尔值
  • 字符
  • 字符串
  • 数组

1 数字

Kotlin 中数字的种类如下：

  • 整数：byte、short、int 、 long
  • 浮点数：float、double

1.1 整型

Kotlin 提供四种不同位数的整型，如下表格：

类型 位数 最小值 最大值
Byte 8 -2^7 2^7 - 1
Short 16 -2^15 2^15 - 1
Int 32 -2^31 2^31 -
Long 64 -2^63 2^63 - 1

Kotlin 支持以下整数字面量形式，但不支持八进制：

  • 十进制：123
  • 十六进制：0x0F
  • 二进制：0b00001011

1.5 数据溢出

数值类型只能表示其定义范围内的值。如果运算结果超出该范围，就会发生溢出。如果你将一个值转换为更小的数值类型，转换后的值可能无法保留原始的数值。这种行为即使编译器接受它，也可能影响你的代码结果。

2 无符号类型

除了一般的整型，Kotlin 还提供一下无符号整数类型：

类型 大小 bit 最小值 最大值
UByte 8 0
UShort 16 0 65,535
UInt 32 0 4,294,967,295 (232 - 1)
ULong 64 0 18,446,744,073,709,551,615 (264 - 1)
